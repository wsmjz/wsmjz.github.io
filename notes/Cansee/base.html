# svg

SVG 渲染器相比 Canvas 渲染器在移动端的总体表现更好 内存占用更低（这对移动端尤其重要）、渲染性能略高、并且用户使用浏览器内置的缩放功能时不会模糊

  1. 路劲动画
    • 参数：M，A, L
